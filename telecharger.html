Télécharger

Téléchargez, le Grasslands Survival Pack ici. Minecraft n'a pas besoin d'être installé sur votre ordinateur. Lisez les instructions avec attention. Si vous avez toujours des soucis, allez voir la section F.A.Q..

Installation

Télécharger, extraire, lancer

  1. Téléchargez l'archive du Grasslands Survival Pack avec les liens plus bas.
  2. Extrayez le dossier de l'installateur se trouvant dedans.
  3. Double-cliquez simplement le fichier "GSP Installateur" dans ce dossier, et suivez les instructions.

L'installateur GSP

  1. En cliquant sur Backup, votre dossier .minecraft sera copié sur votre bureau, dans un dossier appelé "Your minecraft backup". Ceci est optionnel.
  2. Cliquez sur "Installer GSP" pour débuter l'installation.
  3. Si Forge échoue à télécharger ses librairies, il vous faut recommencer toute l'installation.
  4. Une fois terminée, vous pouvez supprimer le dossier "DKF's GSP - Installateur FR" et garder simplement le racourci "DKF's GSP" qui doit avoir apparu sur votre bureau.

Comment jouer

Lancer une partie

  1. Cliquez sur "Play" pour lancer Minecraft et le serveur en cours du manageur dans une console noire (clic droit n'en démarrer que l'un des deux) (si le serveur ne démarre pas, allez voir ICI).
  2. Sélectionnez le profile Forge dans le launcher de Minecraft.
  3. Dans le menu principal, allez dans MULTIJOUEUR et vous devriez pouvoir vous connecter au premier serveur.

    4. Plus

  4. Vous pouvez arrêter le serveur en tapant "stop" dans la console noire.
  5. Quand vous faites "stop" dans un serveur, il se sauvegarde automatiquement, vous pouvez quitter le manageur sans utiliser le bouton "sauver". Il sert simplement à faire une copie du monde en cours, pour que vous puissiez le recharger plus tard, depuis votre "Liste".
  6. Vous pouvez créer un nouveau monde et choisir la taille de ses biomes en appuyant que le bouton rouge : "reset" (cela supprime le monde courant).
  7. Vous pouvez effacer/renommer vos sauvegardes avec un clic droit.
  8. Il est possible d'installer Optifine et le shader mod facilement dans "Options". Hautement recommandé et déjà configuré.

Le plugin World Border

Pré-remplissez votre monde !

Cette fonction peut être très utile si vous ne voulez pas jouer dans l'immédiat. (Vous devez attendre 5 à 20 minutes selon la taille du terrain que vous souhaitez générer)
Il vous permet de pré-générer le terrain avant de jouer, pour que l'ordinateur qui héberge le serveur n'ait pas à la faire en même temps que vous jouez.

Commandes

Dans la console du serveur, tapez les commandes suivantes :
Premièrement : "/wb world set X"
Cela crée une bordure sur le serveur qui a un rayon de X blocs. Plus le rayon est grand, plus le temps de remplissage sera long.
Ensuite : "/wb world fill"
Cette commande démarre la création des chuncks du monde comme si vous l'exploriez.
Pour enlever la bordure : "/wb world clear"
Pour que le plugin arrête de remplir : "/wb fill cancel"

Je recommande de redémarrer le serveur après le remplissage. Vous pouvez le faire tout en jouant, mais si votre ordinateur peut faire ça sans lagger, vous n'avez probablement pas besoin de remplir votre monde avec le plugin.

Pour connaître toutes les commandes, allez sur la page Bukkit de WorldBorder
