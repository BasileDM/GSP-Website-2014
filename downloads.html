Download

Here, you can download the Grasslands Survival Pack. You don't need to have Minecraft installed on your computer. Be sure to read the instructions carefully. If you still have any trouble, go check out the F.A.Q. section.

Installation

Download, extract, launch

  1. Download the Grasslands Survival Pack archive using the links below.
  2. Extract the installer folder that is inside.
  3. Just double click the "GSP Installer" file inside this folder to launch the installer, and follow the instructions.

The GSP Installer

  1. By clicking Backup button, your .minecraft folder will be copied on your desktop, in a folder named "Your Minecraft backup". This is optionnal.
  2. Click "Install GSP" to start the installation.
  3. If Forge fails downloading libraries, you have to start all the installation over again.
  4. When it is completed, you can delete the "DKF's GSP - EN Installer" folder and just keep the "DKF's GSP" shortcut that should have appeared on your desktop.

How to play

Launching a game

  1. Click the "Play" button to launch Minecraft and the current server of the manager in a black console (Right click to start only one of them) (If the server does not start go check out here).
  2. Select the Forge profile in the Minecraft launcher.
  3. In the main menu, go in MULTIPLAYER and you should be able to connect to the first server.

    4. More

  4. You can stop the server by entering "stop" in the black console.
  5. When you "stop" a server, it is automatically saved, you can quit the manager without using the save button. It is only used to make a copy of the current world, so you can load it back later, from your save list.
  6. You can create a new world and choose the size of the biomes by clicking the red "reset" button (deletes the current world).
  7. You can delete/rename your saved server worlds by right clicking them.
  8. You can install Optifine and the shader mod easily in "Options". It is highly recommended and already configured.

World Border plugin

Pre-fill your world !

This function can be really useful if you don't want to play immediately (You have to wait between 5 to 20 minutes depending on the size of the terrain you want to generate).
It allows you to pre-generate the terrain before playing, so that the computer which is hosting the server don't have to do it at the same time you're playing.

Commands

In your server console, you just have to type the following commands :
First : "/wb world set X"
That sets a border on the server which has a radius of X. The bigger the radius, the longer it will take.
And then : "/wb world fill"
That will start filling the chunks of your world as if you were exploring.
To remove the border : "/wb world clear"
To stop the plugin from filling your world : "/wb fill cancel"

I recommend restarting the server after the filling is complete. You can also do it at the same time you are playing, but if your computer supports that without lagging, you probably don't need to fill your world with the plugin.

If you want to know all the commands, go to the WorldBorder Bukkit page
