Forge a échoué

Si quand vous cliquez sur "Install Forge", Forge affiche un message d'erreur, il vous faut recommencer toute l'installation de GSP. Désolé, celà arrive assez rarement et l'installation est assez rapide. Le problème viens des serveurs de Forge et on ne peut rien y faire.

Minecraft 1.6.4 n'est pas détecté

Avez vous bien choisi "Release 1.6.4" dans "Editer profile" ? Effectuez bien chaque étape de l'installation comme décrite.

"Mes documents" s'ouvrent

Si l'installateur ouvre le dossier "Mes documents" au lieu d'un dossier mod ou shader, vous n'avez pas installé GSP correctement. Suivez bien les étapes d'installation.

Le serveur ne se lance pas

Si votre serveur ne se lance pas, plusieurs explications sont possibles. Parfois votre mémoire peut être basse. Fermez un maximum d'applications, puis ré-essayez plusieurs fois. Redémarrez votre ordinateur si vous pensez qu'il le faut.

Une autre possibilité plus courante : votre version de Java n'est pas installée sur le chemin 64 bits. Dans le manageur, allez dans le menu "options" menu, cliquez sur le logo Java et sur "32 bits".

Minecraft ne se lance pas

Si Minecraft crash ou retourne au launcher au lancement, essayez de ré-installer GSP.

Mon jeu crash

Si le jeu ne fait que crasher, soyez certains de ne pas avoir ajouté ou mis à jour un mod. Optifine peut parfois mener à un crash si le terrain se charge trop lentement. Vous avez peut-être installé un mod client (comme Optifine ou le shader mod) sur le serveur.

Je ne peux que "Jouer Hors-ligne"

Dans le launcher Minecraft, si vous n'avez que l'option "Jouer Hors-ligne", ceci est totalement indépendant de GSP. Les serveurs de Minecraft ont des problèmes ou votre connexion est limitée. Essayez ce site web pour savoir si les serveurs Minecraft.net ne répondent plus.

Mon skin est démembré

Celà arrive lorsque vous changez de shader pack. Relancez le jeu et ça devrait être bon.

Je ne peux boire nulle part

Soyez certains de viser le côté immergé d'un bloc solide quand vous cliquez-droit. Votre main doit être vide également. Tuto sur la soif

Je ne peux pas crafter le...

Si vous ne pouvez pas crafter quelque chose, regardez la liste des "Recettes interdites/spécifiques" dans le guide de survie GSP.

Il n'y a pas d'animaux

Avez-vous lancé le jeu jusqu'au menu principal en vanilla 1.6.4 ? C'est une étape importante de l'installation.

Mes vaches (ou autre) ont disparues

Avez-vous posé des torches aux environs ? C'est spécifié dans la section "Élevage" du guide de survie.

Je lag beaucoup

Si vous avez des lags, vous pouvez faire certaines choses :

Pour ceux qui jouent en solo, allez dans "Options" dans le manageur GSP et cliquez sur "Dossier serveur". Celà ouvre le dossier du serveur en cours. Ouvrez le fichier "server.properties" avec n'importe quel éditeur de texte et changez la ligne "ip=" en "ip=127.0.0.1". Les autres joueurs ne pourront pas vous rejoindre, mais les lags devraient être minimisés. Vous pouvez aussi faire un clic droit sur "Dossier serveur" et cliquer sur "Ouvrir serveur source". Si vous changez quelque chose ici, tous les nouveaux serveurs que vous allez créer en faisant "Reset" conserveront ces changements.

Redémarrer le serveur toutes les 30 minutes est recommandé et devrait enlever le lag pendant un moment. Si vous explorez, la génération de terrain cause beacoup de lags, c'est pour celà que j'ai implémenté "World Border" pour charger leterrain avant de jouer grâce à sa fonction /fill.

La génération du terrain est lente (et buggée ?)

Essayez de supprimez Optifine, qui ralentit la génération du terrain. Essayez également la fonction "/fill" du plugin "World Border" (Tutorial here). Pensez aussi à faire "F3 + A" pour rafraîchir le terrain !

Mes amis me voient buggé

Lorsque vous mourrez, les choses suivantes se produisent : Vous n'avez que 6 coeurs au début mais après être mort, vous aurez tous vos coeurs à nouveau. La plupart d'entre eux seront vides bien sûr, et vous ne pourrez pas vous régénéreer entièrement. Vous ne serez plus ralentit quand vous aurez peu de vie et les autres joueurs vous verront voler, et buggé. Pour tout résoudre, il suffit de vous déconnecter/reconnecter.

Une autre possibilité est que vos amis ont changé de shader pack. Il leur faut relancer le jeu.
